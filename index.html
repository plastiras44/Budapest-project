Tetris Wannabe

A chaotic co-op twist on classic Tetris.

About the Game

Tetris Wannabe is a PC game developed as part of an academic assignment, built in just four days to practice and improve our game development skills. Inspired by classic Tetris, the game focuses on fast-paced gameplay and cooperative chaos.

How to Play

  • Control falling blocks to clear lines
  • Coordinate with other players
  • Survive as long as possible
  • Adapt to unexpected gameplay changes

The Twist

The game introduces a unique gameplay mechanic that changes how players interact with the board and with each other. This twist is designed to create intense, unpredictable moments during play.
